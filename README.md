# Weather App

A simple Weather App built using HTML, CSS, and JavaScript.

## Features
- Search weather by city name
- Displays temperature in Celsius
- Shows humidity information
- Displays weather conditions
- Clean and responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## Project Structure
weather-app/
├── index.html
├── index.css
└── index.js

## How to Use
1. Enter a city name.
2. Click the Search button.
3. View the current weather details.
